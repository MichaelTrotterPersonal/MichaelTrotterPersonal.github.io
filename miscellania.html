Bad CSS changes

The pages on this site are viewable with two different CSS styles applied.

The first, which you will see by default when loading a page, has been styled for accesibility and aesthetics.

The second, which you can access by clicking the bright green "BadCSS" link at the bottom of each page, has been created as a demonstration of poor design choices. Shown below is the embedded bad.css which is applied when you click the BadCSS link. In the top section it includes all the changes that have been implemented to make the page less aesthetically pleasing and less accessible, along with short comments explaining those changes.
